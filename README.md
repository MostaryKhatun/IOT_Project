# IoT Projects Collection

This repository contains four IoT projects developed using **ESP32**, **Wokwi Simulator**, and **Blynk Cloud**. These projects demonstrate different Internet of Things (IoT) applications including home automation, smart healthcare, lighting control, and study assistance.

---

## Repository Contents

### 1. Smart Medicine Reminder

A smart medicine reminder system that reminds users to take medicine on time using ESP32, OLED display, buzzer, servo motor, and Blynk Cloud.

### Features

- ESP32 based system
- Wi-Fi connectivity
- Blynk mobile application
- OLED display notifications
- Medicine reminder alarm
- Servo-controlled medicine box
- Automatic reminder scheduling

---

### 2. Smart Home Automation

A home automation project for controlling household appliances remotely using ESP32 and the Blynk IoT platform.

### Features

- Remote appliance control
- Wi-Fi communication
- Blynk Cloud integration
- Relay control
- Real-time monitoring

---

### 3. RGB LED Control

An IoT project for controlling RGB LEDs using the Blynk mobile application.

### Features

- ESP32 Wi-Fi control
- RGB color control
- Mobile application interface
- Real-time LED adjustment
- Easy user interaction

---

### 4. Smart Exam Study Assistant

An IoT-based study assistant that helps students manage study sessions and break times.

### Features

- Study timer
- OLED display
- RGB status indicator
- Buzzer notification
- Blynk mobile control
- ESP32 based implementation

## How to Use

1. Download the desired project ZIP file.
2. Extract the ZIP file.
3. Open the Arduino project.
4. Install the required libraries.
5. Update your Wi-Fi credentials.
6. Update the Blynk Authentication Token.
7. Upload the code to ESP32.
8. Open the Blynk application.
9. Run the project.
